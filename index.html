sessionStorage , localStorage 区别

localStorage

永久存储于本地电脑磁盘中,关闭浏览器、电脑、断电,数据都在

可存储大小为20Mb

可以多窗口数据共享 ( 但是遵守同源策略 )

sessionStorage

临时存储,当页面关闭的时候,session数据会被自动清空

可存储大小为5Mb

不能多窗口数据共享 ( 通过跳转可以 )

页面跳转可以通过session实现数据共享

改变主轴方向

项目默认水平排列

flex-direction: row(主轴水平,左-->右) | row-reverse (主轴水平方向翻转右-->左) | column(主轴垂直 上-->下) | column-reverse (主轴垂直反转 下-->上)

控制元素在主轴的对齐方式

justify-content : flex-start(以起点对齐) | flex-end(以终点对齐) | center(居中对齐) | space-between(两端对齐) | space-around(环绕对齐)

对齐永远是整体对齐,不会颠倒项目次序,而flex-dircetion会改变项目次序

/*flex-wrap : 控制项目是否换行*/
flex-wrap : no-wrap(默认,不换行) | wrap(换行) | wrap-reverse(颠倒换行(换行时向上换))

/*flex-flow : flex-diection和flex-wrap的简写*/

align-item

侧轴(交叉轴)单行情况下的对齐方式

align-item : flex-start (起点对齐) | flex-end (终点对齐) | center (居中对齐) | stretch (默认值,没有高度时自动将项目高度和容器高度设置为一致)

align-content

定义多行轴线的对齐方式,如果轴线只有一行,那么这个属性不起作用

align-content : flex-start | flex-end | center(居中对齐) | space-between(两端对齐) | space-around(环绕对齐) | stretch(拉伸)

线性渐变

background-image: linear-gradient(to bottom, color);

linear-gradient 控制的是线性渐变

参数:

  1. 参数1. 渐变线的角度: 可以用 to 方位名词 / 角度deg(默认180deg,从上到下) ,方向是按照顺时针递增
  2. 参数2. 颜色,可以逗号分隔多个颜色值

  3. 每个颜色后面可以加百分比,颜色会延伸至此(止色位置)

    径向渐变

    background: radial-gradient();

  4. 参数1 圆的大小,一个值是正圆,两个值是椭圆

  5. 参数2 圆心位置, at x坐标 y坐标 / at 方位名词 / at 百分比
  6. 参数3 颜色,可以像线性渐变一样加百分比

    浏览器前缀

  7. 目的:兼容低版本高级浏览器
  8. -webkit- 苹果, 谷歌
  9. -moz- 火狐
  10. -o- opera
  11. -ms- IE

    2D转换

    元素在2D平面上的移动,旋转,缩放,斜切,称为2D转换

键名:transform :

键值:

  1. translate( ) 位移,一个参数控制水平移动,两个参数控制水平和垂直,单位可以是 px, 或者百分比(参照的是元素自身宽高)
  2. translateX(), translateY() 单独移动一个方向,一个参数
  3. translate中的参数指的是位置,而不是移动距离

扩展

  1. 绝对定位( top : 50% ; left : 50% )配合 translate( -50%, -50% ) 可以帮助元素居中,不写明宽度时,元素宽度最多只有父元素宽度的一半(因为left 50%,而高度有滚动条存在)
  2. translate 类似于相对定位

旋转 rotate

translate: rotate( 45deg );
transform-origin: center center;
transform-origin: 100px 100px;
transform-origin: 50% 50%;

缩放 scale

transform: scale(1,1);
transfrom: scaleX(1),scaleY(1);
  1. 参数是倍数,没有单位的数值,可以是小数,可以是一个参数,可以是两个参数分别控制

斜切 skew

transfrom: skew(10deg);
transform: skewX(10deg);
transform: skewY(10deg);
transform: skew(10deg,10deg);

X方向斜切,中心点控制上下;

Y方向斜切,中心点控制左右.

多个2D效果共存

  1. 多个效果共存需要写在一行上
  2. 在不同场景出现transfrom时,需要复制写上之前已经存在在的属性,防止覆盖和冲突
  3. transfrom转换顺序尽量保持统一,如果不统一可能出现问题

    过渡 transition

    transition: width 2s, height 2s;
    transition: all 2s;
  4. 参数说明
    1. 发生过渡的属性 (写成all 相对来说浪费性能)
    2. 过渡的时间 单位是 ms 和 s
    3. 过渡曲线
    4. 过渡延时 delay
  5. 目前display 和 渐变不能直接产生过度
  6. 过渡的属性最好写全
