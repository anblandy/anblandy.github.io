HTML5-day03

线性渐变

background-image: linear-gradient(to bottom, color);

linear-gradient 控制的是线性渐变

参数:

  1. 参数1. 渐变线的角度: 可以用 to 方位名词 / 角度deg(默认180deg,从上到下) ,方向是按照顺时针递增
  2. 参数2. 颜色,可以逗号分隔多个颜色值

  3. 每个颜色后面可以加百分比,颜色会延伸至此(止色位置)

    径向渐变

    background: radial-gradient();

  4. 参数1 圆的大小,一个值是正圆,两个值是椭圆

  5. 参数2 圆心位置, at x坐标 y坐标 / at 方位名词 / at 百分比
  6. 参数3 颜色,可以像线性渐变一样加百分比

    浏览器前缀

  7. 目的:兼容低版本高级浏览器
  8. -webkit- 苹果, 谷歌
  9. -moz- 火狐
  10. -o- opera
  11. -ms- IE

    2D转换

    元素在2D平面上的移动,旋转,缩放,斜切,称为2D转换

键名:transform :

键值:

  1. translate( ) 位移,一个参数控制水平移动,两个参数控制水平和垂直,单位可以是 px, 或者百分比(参照的是元素自身宽高)
  2. translateX(), translateY() 单独移动一个方向,一个参数
  3. translate中的参数指的是位置,而不是移动距离

扩展

  1. 绝对定位( top : 50% ; left : 50% )配合 translate( -50%, -50% ) 可以帮助元素居中,不写明宽度时,元素宽度最多只有父元素宽度的一半(因为left 50%,而高度有滚动条存在)
  2. translate 类似于相对定位

旋转 rotate

translate: rotate( 45deg );
transform-origin: center center;
transform-origin: 100px 100px;
transform-origin: 50% 50%;

缩放 scale

transform: scale(1,1);
transfrom: scaleX(1),scaleY(1);
  1. 参数是倍数,没有单位的数值,可以是小数,可以是一个参数,可以是两个参数分别控制

斜切 skew

transfrom: skew(10deg);
transform: skewX(10deg);
transform: skewY(10deg);
transform: skew(10deg,10deg);

X方向斜切,中心点控制上下;

Y方向斜切,中心点控制左右.

多个2D效果共存

  1. 多个效果共存需要写在一行上
  2. 在不同场景出现transfrom时,需要复制写上之前已经存在在的属性,防止覆盖和冲突
  3. transfrom转换顺序尽量保持统一,如果不统一可能出现问题

    过渡 transition

    transition: width 2s, height 2s;
    transition: all 2s;
  4. 参数说明
    1. 发生过渡的属性 (写成all 相对来说浪费性能)
    2. 过渡的时间 单位是 ms 和 s
    3. 过渡曲线
    4. 过渡延时 delay
  5. 目前display 和 渐变不能直接产生过度
  6. 过渡的属性最好写全
